Datenschutzerklärung

Stand: Februar 2026

1. Datenverarbeitung auf einen Blick

Der Schutz Ihrer Gesundheitsdaten ist uns ein wichtiges Anliegen. Diese App wurde so konzipiert, dass sie keine persönlichen Daten auf externen Servern speichert.

2. Erhebung von Gesundheitsdaten (EKG & Puls)

Die App empfängt über Bluetooth Daten von Ihrem Polar H10 Brustgurt.

  • Lokale Speicherung: Alle Messungen werden ausschließlich lokal auf Ihrem Smartphone gespeichert.
  • Kein Cloud-Sync: Es findet keine automatische Übertragung an Server oder Drittanbieter statt.
  • PDF-Export: Wenn Sie die PDF-Export-Funktion nutzen, wird die Datei lokal auf Ihrem Gerät erstellt. Sie haben die volle Kontrolle darüber, ob und mit wem Sie diese Datei teilen.

3. Berechtigungen

Die App benötigt Zugriff auf Bluetooth (Scan & Connect), um eine Verbindung zum Sensor herzustellen. Ohne diese Berechtigung kann die Kernfunktion der App nicht genutzt werden.

4. Ihre Rechte

Da alle Daten lokal auf Ihrem Gerät liegen, können Sie diese jederzeit durch Löschen der App oder innerhalb der Tagebuch-Funktion der App selbstständig entfernen.
